#### 安装

```shell
git clone https://github.com/kayw-geek/php-wifi.git
```

#### CLI使用

```shell
php wifi --scan
php wifi --connect <ssid> --password <password>
php wifi --info <ssid>
php wifi --disconnect
php wifi --scan
```
